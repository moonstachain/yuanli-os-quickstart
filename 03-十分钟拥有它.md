# 3 · 十分钟拥有它

好消息——我们把装机做成了**一行命令**。你不用再一步步装 Ollama、不用懂代码，**复制一行、粘贴、回车、去喝杯咖啡**，回来你的第二大脑就装好了。

> 🚀 **更简单的"双击就装"也在路上**。在那之前，下面这一行已经把绝大部分麻烦替你自动化了——它会自动帮你装好一切（基础工具、本地思考核心、AI 模型、大脑本体）。

---

## 你只需要准备一样东西

**一台苹果 Mac 电脑**——你自己的。

为什么必须是你自己的电脑？因为**你的资料只存在这台电脑里，绝不上云**（这是为你的数据安全，第 6 章细说）。就这一个要求，别的都交给那一行命令。

> Windows 电脑暂时还不支持，我们在做了。

---

## 就这一行（复制 → 打开"终端" → 粘贴 → 回车）

**第一步**：在 Mac 上打开自带的**"终端"**小程序。
（最快的方法：按 `⌘ + 空格`，输入"终端"两个字，回车。）

**第二步**：把下面这一整行复制进去，按回车：

```bash
curl -fsSL https://raw.githubusercontent.com/moonstachain/yuanli-os-quickstart/main/install.sh | bash
```

**第三步**：**去喝杯咖啡。** ☕

它会自己一步步装好，全程约 30-45 分钟（大部分时间在下载）。中间：

- 如果它弹出一个苹果的小窗口让你装"命令行工具"——**点一下【安装】**就行（一次性，以后不用再点）。
- 其余全自动，你不用管。屏幕上会用大白话告诉你进行到哪了。

---

## 怎么知道装好了？

装完，终端里会出现一个大大的：

> 🎉 **装好了！你的第二大脑已经就位**

想再确认一下，在终端输入这一句、回车：

```bash
yos status
```

**看到一排绿色的勾**，就说明一切就位了。🎉

## 万一卡住了？

- 最常见是**网络问题**——换个网络，或者**把同一行命令再粘一次**（它很聪明，装过的步骤会自动跳过，接着没装完的继续）。
- 实在搞不定，把终端里红色的字截图，发给你的技术伙伴或我们团队。
- 定心丸：**装坏了重来也不会丢数据**（你的资料存在文件里，第 8 章细说）。
