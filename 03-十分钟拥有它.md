# 3 · 十分钟拥有它

先说句实话：**这是整个过程里唯一需要一点点技术的地方。** 但别担心——装一次，以后就再也不用碰了，而且我们给你两条路。

> 🚀 **好消息（马上会更简单）**：我们正在做"双击一下就装好"的一键安装包。如果你不急，可以等它；如果你想现在就用，下面两条路都很快。

---

## 你需要先准备两样东西

1. **一台 Mac 电脑**（你自己的，因为你的资料只存在这台电脑里，不上云——这是为了你的数据安全，第 6 章会讲）。
2. **装一个免费小软件叫 Ollama**（让你的大脑能在本地思考，不把你的资料发出去）。去 [ollama.com](https://ollama.com) 下载、双击安装，跟装任何 App 一样。

---

## 路线 A：交给你的助理或技术伙伴（推荐，最省心）

把下面这段话**原封不动**发给你的助理、技术伙伴，或任何会用电脑命令行的人。对他们来说是 10 分钟的事：

```
帮我在这台 Mac 上装一个叫"原力OS"的第二大脑，跟着做：
1. 确认已装 Ollama（ollama.com）并打开
2. 打开"终端"，粘贴这两行，回车：

git clone https://github.com/moonstachain/yuanli-strategy-soul.git ~/AI\ Project/yuanli-strategy-soul
bash ~/AI\ Project/yuanli-strategy-soul/scripts/yos install --role colleague

3. 等它自己跑完（约 30-60 分钟，中间不用管）
4. 跑完后输入 yos status，看到一排绿色的勾就成了
```

装好后，把电脑还给你，你就可以开始用了。

---

## 路线 B：自己动手（不用懂代码，照抄就行）

如果你想自己来，其实也就是**复制粘贴**：

1. 装好并打开 **Ollama**（[ollama.com](https://ollama.com)）。
2. 在 Mac 上找到并打开**"终端"**这个自带的小程序（在"应用程序 → 实用工具"里，或用聚焦搜索输入"终端"）。
3. 把这两行**复制进去、按回车**：

```bash
git clone https://github.com/moonstachain/yuanli-strategy-soul.git ~/AI\ Project/yuanli-strategy-soul
bash ~/AI\ Project/yuanli-strategy-soul/scripts/yos install --role colleague
```

4. 然后就**去喝杯咖啡**——它会自己下载、安装、准备好，约 30-60 分钟，中间不用你管。
5. 如果中途它停下来提示你缺点什么（比如让你先登录一下），照着提示做完，**再把同一行粘贴一次**就会接着跑（它很聪明，做过的步骤会自动跳过）。

---

## 怎么知道装好了？

装完，在终端输入这一句、回车：

```bash
yos status
```

**看到一排绿色的勾**（引擎、大脑、状态都是 ✓），就说明你的第二大脑已经就位了。🎉

以后你也可以随时敲这句，检查它是不是好好的。

---

## 万一卡住了？

- 最常见的是网络问题——换个网络、或稍等重试同一行命令。
- 实在搞不定，把终端里红色的报错截图，发给你的技术伙伴，或联系我们团队。
- 记住一条定心丸：**你的资料存在文件里，装坏了重装也不会丢**（第 8 章细说）。
