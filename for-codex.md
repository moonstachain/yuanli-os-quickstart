# 原力OS 第二大脑 · Codex 执行包

> 这份文件有两部分：**Part A 给 Codex 读**（AI agent 照着自动完成装机+接入+验证），**Part B 给用你 Codex 的同事读**（怎么启动、怎么授权、装完怎么用、要反馈什么）。

---

# Part A · 给 Codex 的执行指令

你是同事的编程助手 Codex。你的任务：在这台 Mac 上，帮同事把「原力OS 第二大脑」装好、接进你自己（Codex）、验证能用，然后教同事怎么用。按顺序做，每步等成功再下一步。

## 步骤 1 · 装机 + 接入（一条命令）

在终端执行：

```bash
curl -fsSL https://raw.githubusercontent.com/moonstachain/yuanli-os-quickstart/main/codex-setup.sh | bash
```

它会自动：确认第二大脑已装（没装就自动装，约 30-45 分钟，中途会下 ~6GB 模型）→ 启动大脑服务（127.0.0.1:3131）→ 把大脑接进 Codex（`gbrain connect --agent codex --install`）→ 把接入凭证写进 shell profile。

- 中途如果弹出苹果的「安装命令行工具」窗口，告诉同事**点一下【安装】**。
- 跑完看到「🎉 接好了」就成功了。

## 步骤 2 · 让接入生效

`codex mcp add` 之后，**当前 Codex 会话加载不了新 MCP**。请让同事：**关掉当前 Codex 会话，重新开一个**。你（新会话的 Codex）启动时才会连上 gbrain。

## 步骤 3 · 自检连通

在新会话里，调用 `get_brain_identity` 确认连上了哪个大脑，再 `search`/`query` 一个话题（如"原力"）确认能查到 soul 正典。查得到 = 接入成功。

## 步骤 4 · ⚠️ 源路由约定（关键，务必遵守）

这个大脑有多个「源」。你帮同事读写时，**必须按用途选对源**：

| 同事想干什么 | 你用哪个工具、哪个源 |
|---|---|
| **记同事自己的判断/客户洞察/决策** | `put_page`，**source = `personal`**（同事的私密空间，装机时自动建好） |
| **查同事自己以前记的东西** | `query`，**source = `personal`** |
| **查原力OS 的正典知识/方法论** | `query`，**source = `soul`**（公开正典） |

- `capture` 是命令行工具、**不是 MCP 工具**——你要写东西，用 `put_page` 到 `personal` 源，别找 capture。
- 给 `personal` 源写页面时，slug 用有意义的短名（如 `personal/客户-XX-判断-2026-07-08`），内容就是同事的原话 + 你补的结构。

## 步骤 5 · 教同事怎么用（然后交还给他）

告诉同事：以后不用敲任何命令，**直接对你（Codex）说话**就行：
- 「帮我记一条：……」→ 你 `put_page` 到 personal
- 「我以前怎么判断……？」→ 你 `query` personal
- 「原力OS 里 X 是怎么说的？」→ 你 `query` soul

# Part B · 给同事的话（人类看这里）

## 你要做的其实很简单

1. **打开你的 Codex**（在这台要装的 Mac 上）。
2. 对 Codex 说这么一句：

   > "读 https://raw.githubusercontent.com/moonstachain/yuanli-os-quickstart/main/for-codex.md 的 Part A，照着帮我装好并接进你自己，装完教我怎么用。装机需要你执行终端命令，我授权你跑。"

3. **授权 Codex 执行终端命令**（它要跑安装脚本）。然后**去喝咖啡** ☕，装机约 30-45 分钟。
4. 中途如果弹出苹果的"安装命令行工具"窗口，**点一下【安装】**。
5. 装完，Codex 会让你**关掉会话、重开一个新的**——照做。
6. 在新会话里，Codex 会自检一下，然后教你怎么用。

## 装完你就能这样用（不用敲命令）

在 Codex 里直接说话：
- 「帮我记一条：刚见的客户，资源是真的，但说话留三分，值得长期处。」
- 「我以前怎么判断这类要长账期的客户？」
- 「原力OS 里，第二大脑是怎么定义的？」

它会自动帮你记进你的私人空间、或把你过去的判断查给你。**你的东西只存在这台电脑，私密、不上云。**

## 装完请把这些反馈发回来（帮我们把产品做得更好）

这是我们第一次让同事用 Codex 装接第二大脑，你的反馈很宝贵：

1. **装机 + 接入总共花了多久？**
2. **中间卡在哪一步了吗？** 有的话把截图发来。
3. **重开 Codex 后，自检（get_brain_identity + 查"原力"）成功了吗？**
4. **试着"记一条、再查出来"顺不顺？** Codex 有没有正确存到你的个人空间？
5. **你的 Codex 版本**（终端敲 `codex --version`）和 **macOS 版本**。

辛苦了！有任何卡点随时找团队。

## 前提 & 说明（技术备注）

- 需要 **codex CLI v0.130+**（脚本会先检测）。
- 接入走 HTTP MCP（`http://127.0.0.1:3131/mcp` + bearer token），复用装机时起的常驻大脑服务。token 是长效全权凭证，只存在你本机 shell env、绝不入任何仓库。
- 这套目前还需要"重开会话生效""对 Codex 说话来记查"——我们正在做**打开 App 就能直接说的更傻瓜版本**，很快见。
